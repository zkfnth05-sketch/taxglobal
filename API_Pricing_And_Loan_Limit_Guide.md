# 국내 신용평가사(NICE / KCB) 및 핀테크 API(쿠콘 / 코드에프) 비용 & 대출 한도 비교 가이드

---

## 1. 💳 NICE 평가정보 (NICE)
국내 1위 신용평가사 (공신력 및 관공서/금융기관 연동률 1위)

- **초기 도입비**: 약 50만 원 ~ 100만 원 *(행사 시 면제 또는 할인)*
- **본인인증 수수료**: 건당 약 25원 ~ 45원 *(SMS/PASS 본인확인)*
- **신용/대출조회 API 수수료**: 월 기본료 10만 원 ~ 30만 원 + 건당 약 50원 ~ 120원
- **특징**: 국내 금융기관 및 관공서 연동률 1위로 공신력이 가장 높음.

---

## 2. 🏛️ KCB 올크레딧 (AllCredit)
NICE와 양대 산맥을 이루는 2대 신용평가사 (토스, 카카오뱅크 등 핀테크 채택률 높음)

- **초기 도입비**: 약 50만 원 ~ 100만 원
- **본인인증 수수료**: 건당 약 25원 ~ 40원
- **신용/대출조회 API 수수료**: 월 기본료 10만 원 ~ 20만 원 + 건당 약 40원 ~ 100원
- **특징**: 핀테크 앱 연동 지원이 잘 되어 있고 대출 승인 모형 조회 시 속도가 빠름.

---

## 💡 NICE / 올크레딧 vs 쿠콘 / 코드에프 사용 목적 비교

| 서비스 | 주요 역할 | 추천 용도 |
| :--- | :--- | :--- |
| **NICE / 올크레딧** | **신용점수, 연체 기록, 기존 대출 총액** 조회 | 고객의 신용등급/점수 확인 및 기본 신용 심사 |
| **쿠콘 / 코드에프** | **여러 은행의 실시간 승인 대출 한도·금리** 비교 | "A은행 2,000만 원 / B저축은행 3,500만 원" 실시간 한도 안내 |

> **💡 표준 핀테크 구축 조합**:
> 보통 대출 비교 플랫폼에서는 **NICE나 올크레딧으로 본인확인 및 신용점수를 1차 확인**하고, **쿠콘 API로 각 은행별 대출 승인 한도를 받아와서 화면에 띄워주는 조합**으로 가장 많이 사용합니다.
